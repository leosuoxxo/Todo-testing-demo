# todo

使用Vue 進行unit test & e2e test

## Todo 情境
1. 應顯示程式標題與欄位
2. 輸入欄位應該在代辦事項輸入完成按下Enter後清空
3. 建立新待辦事項後應該出現待辦事項列表
4. 當沒有待辦事項時列表應該隱藏
5. 當完成待辦事項後應標記為已完成
6. 應可從列表上刪除待辦事項
